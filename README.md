## Core Positioning

**ODE is the AI-native engineering platform** - not a legacy tool with AI bolted on.

### Key Differentiators
1. AI-native from day one (vs. retrofitted AI in traditional CAD/CAE)
2. Unified MBSE + CAD + CAE in browser (no tool silos)
3. Modelica + LLM synergy (equation-based = machine-friendly)
4. Cloud-native, zero-install, real-time collaboration
5. Open standards (Modelica 4.0, FMI 2.0/3.0, SysML v1.6)

## Business Model

- **SaaS Subscription**: Tiered plans (Free / Starter / Pro / Team / Enterprise)
- **AI Token Consumption**: Pay-per-use AI model generation and analysis
- **Cloud Compute**: On-demand FEA/CFD/simulation compute billing
- **Storage & Collaboration**: Team-based project storage

## Target Segments

1. **Aerospace/Automotive OEMs** - Enterprise, high ACV, SLG motion
2. **Engineering Consultancies** - Mid-market, use-case driven
3. **Universities/Research** - Community building, future champions
4. **Independent Engineers/Makers** - PLG funnel, high volume
