# ODE 12-Month Launch Plan

## Overview

Five-phase launch roadmap to reach **$1M ARR** within 12 months.

---

## Phase A: Foundation (Month 1-2)
**Budget: $75,000 | Target: Brand live, website launched, first 200 free users**

### Objectives
- Complete brand refresh and new visual identity
- Launch redesigned website with AI demo
- Establish content engine (blog, social)
- Set up PLG infrastructure (free tier, onboarding)

### Key Activities
1. Finalize brand identity, voice guide, visual system
2. Build and launch new website (dark mode, AI demo embed)
3. Set up analytics (Mixpanel), CRM (HubSpot), email (Customer.io)
4. Publish 8 foundational blog posts (SEO pillars)
5. Launch LinkedIn content program (founder + company)
6. Create 3 interactive demo scenarios
7. Set up free tier with automated onboarding flow
8. Begin Google Ads and LinkedIn Ads testing ($2K/mo)

### Deliverables
- [ ] New website live at orthogonal.dev
- [ ] Brand guidelines document published
- [ ] 3 AI demo scenarios available
- [ ] CRM and analytics configured
- [ ] 8 blog posts published
- [ ] Paid ad campaigns running

### Success Metrics
| Metric | Target |
|--------|--------|
| Website visitors | 2,000/month |
| Free sign-ups | 200 cumulative |
| Paid users | 10-15 |
| MRR | $3,300 |

## Go/No-Go Decision Points

| Decision | When | Criteria |
|----------|------|----------|
| Scale paid spend | M3 | CAC <$500 blended, >2% sign-up conversion |
| Commit to enterprise sales | M4 | >2 pilot requests, product feedback positive |
| Begin Series A prep | M8 | >$50K MRR, >8 enterprise accounts |
| Hire first AE | M10 | >$60K MRR, sales pipeline >$500K |
