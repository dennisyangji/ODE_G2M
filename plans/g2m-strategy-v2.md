> Source PRD: `gtm/prd-g2m-strategy-v2.md`  
> Date: 2026-04-12  
> Execution Model: **All phases run in parallel. AI-era company = maximum concurrency.**

---

## Architectural Decisions

Durable decisions that apply across all phases:

- **Business model**: Token-first (AI compute + LLM usage billed per token) + Services (Professional Services packages). Subscriptions are the access vehicle; tokens are the revenue engine.
- **AI-native operations**: Every team member runs AI agents as digital workforce. Standard: 1 person = 3–5 AI workers. No manual task that an agent can do.
- **Brand**: Orthogonal (parent) = "the Dassault of the AI era." ODE = flagship product.
- **Design system**: Apple-inspired — SF Pro, `#F5F5F7` backgrounds, `#1D1D1F` text, radical whitespace, cinematic renders. Applied to all public surfaces.
- **GTM motion**: PLG (free token allowance → upgrade) + SLG (enterprise service packages) + Channel (reseller revenue share 20–30%).
- **Tool release**: Non-blocking. Each tool ships independently when ready; GTM runs ahead of product on positioning.

## Phase 2: Token + Services Business Model

**User stories**: #3, #15

### What to build

Restructure all commercial materials so **tokens are the primary product**, not subscriptions. Subscriptions become the access tier; the real revenue is token consumption and service packages.

**Token Model (Product Revenue):**

| Tier | Access | Token Bundle | Overage |
|------|--------|-------------|---------|
| Free | Core tools | 50K tokens/month | Upgrade prompt |
| Starter $49/mo | All tools | 500K tokens/month | $0.03/1K |
| Pro $99/mo | All tools + API | 5M tokens/month | $0.02/1K |
| Team $199/seat/mo | All tools + admin | 20M tokens/seat/month | $0.015/1K |
| Enterprise | Custom | Dedicated token pool | Negotiated |

Tokens cover: AI model generation, AI simulation setup, AI results interpretation, AI documentation, AI optimisation runs, AI co-pilot queries.

**Services Model (High-Margin Revenue):**

| Service Package | Price | Delivery |
|----------------|-------|---------|
| Onboarding Sprint | €5,000 | 2-week guided setup by AI-assisted team |
| Integration Pack | €10,000–€25,000 | FMI/PLM integration (Teamcenter, Windchill) |
| Custom Solver | €20,000–€50,000 | Domain-specific solver deployment |
| AI Training Data | €15,000–€40,000 | Fine-tune ODE AI on customer's engineering data |
| Retainer (CSM+) | €3,000/month | Dedicated AI-assisted customer success |

**TCO Calculator (Sales Tool):**
Build a "Dassault vs. ODE" TCO comparison tool:
- Input: seats, disciplines needed, current toolchain
- Output: 3-year cost comparison, token usage projection, service ROI

**Acceptance criteria:**
- [ ] Pricing page rebuilt: token bundles are the hero, subscriptions are the access wrapper
- [ ] "Token calculator" on pricing page — user inputs usage, sees monthly cost
- [ ] Services page live with 5 packages, clear scope and pricing
- [ ] "Dassault vs. ODE TCO" one-pager built (AI-generated, designer-polished)
- [ ] Sales deck updated: token model explained in ≤3 slides, services pipeline shown

## Execution Principles

> **AI-era company = maximum concurrency. No phase waits for another.**

1. **Parallel by default** — all 7 phases run simultaneously. Each has its own owner and AI agent stack.
2. **AI drafts, humans approve** — no human should write a first draft of anything. AI generates, human refines.
3. **Ship fast, iterate faster** — a landing page live in 2 days beats a perfect page in 2 weeks.
4. **Measure everything with AI** — weekly AI-generated metrics report across all phases, surfaced to CEO in 5 minutes.
5. **1 person = 3–5 AI workers** — tracked as a KPI. If someone is working without AI, ask why.
